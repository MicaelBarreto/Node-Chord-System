Chord System

This is a simplified Chord System that starts with 40 nodes that represents musics but only 8 nodes start active.

You can add and delete nodes and each have the following structure:

  • Node ID
  • Music Name
  • MAC Address
  • Status (Active/Desactive)
  • Next Node

You can navigate between screens by following the file button on the top left of the window

This program was made by Micael Moraes using NodeJS, ElectronJS and SQLite
